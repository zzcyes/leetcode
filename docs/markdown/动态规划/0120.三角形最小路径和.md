# 120.三角形最小路径和

| Title | Describe                                                           |
| :---- | :----------------------------------------------------------------- |
| 题目  | [120.三角形最小路径和](https://leetcode-cn.com/problems/triangle/) |
| 难度  | ⭐⭐                                                               |

## 题目

给定一个三角形，找出自顶向下的最小路径和。每一步只能移动到下一行中相邻的结点上。

相邻的结点 在这里指的是 下标 与 上一层结点下标 相同或者等于 上一层结点下标 + 1 的两个结点。

例如，给定三角形：

```
[
     [2],
    [3,4],
   [6,5,7],
  [4,1,8,3]
]
```

自顶向下的最小路径和为  11（即，2 + 3 + 5 + 1 = 11）。

说明：

如果你可以只使用 O(n)  的额外空间（n 为三角形的总行数）来解决这个问题，那么你的算法会很加分。

## 题解

### 方法一：动态规划（自顶向下做状态转移，考虑边界）

#### 思路

将题目给定的三角形变成等腰三角形展示：

```
[
    [2],
    [3,4],
    [6,5,7],
    [4,1,8,3]
]
```

##### 1. 确定 dp 含义

dp[i][j]表示三角形顶部走到位置(i,j)的最小路径和。(i,j)为三角形中第 i 行第 j 。

根据题干：相邻的结点 在这里指的是 下标 与 上一层结点下标 相同或者等于 上一层结点下标 + 1 的两个结点。

##### 2.建立 dp 关系(建立状态转移方程)

那么走到位置(i,j),上一步只能在位置(i-1,j-1)或(i-1,j)。可以得出状态转移方程：

```
dp[i][j] = Math.min(dp[i-1][j-1] , dp[i-1][j]]) + triangle[i][j];
```

到这里还没结束，triangle 数组是表示一个三角形，那么，当 j=0 时，dp[i-1][j-1]没有意义，

即当我们在第 i 行的最左侧时，我们只能从第 i-1 行的最左侧移动过来。因此状态转移方程为：

```
dp[i][0]=dp[i−1][0]+triangle[i][0]
```

当 j=i 时，dp[i-1][j] 没有意义，即当我们在第 i 行的最右侧时，我们只能从第 i-1行的最右侧移动过来。因此状态转移方程为：

```
dp[i][i]=dp[i−1][i−1]+triangle[i][i]
```

##### 3. 确定边界值。

最后还要注意边界条件，即在三角形的顶部时，最小路径和就等于对应位置的元素值，那就是:

```
dp[0][0] = triangle[0][0]
```

最终的答案即为 dp[n-1][0] 到 dp[n-1][n-1] 中的最小值，其中 n 是三角形的行数。

#### 代码

```javascript
/**
 * @param {number[][]} triangle
 * @return {number}
 */
var minimumTotal = function(triangle) {
    let n = triangle.length;
    let dp = new Array(n).fill().map((item,index)=>new Array(index+1));
    // 边界情况
    dp[0][0] = triangle[0][0];

    for (let i = 1; i < n; ++i) {
        // j=0 时，dp[i-1][j-1]没有意义
        dp[i][0] = dp[i - 1][0] + triangle[i][0];

        for (let j = 1; j < i; ++j) {
            // 正常情况
            dp[i][j] = Math.min(dp[i - 1][j - 1], dp[i - 1][j]) + triangle[i][j];
        }

        // j=i 时，dp[i-1][j] 没有意义
        dp[i][i] = dp[i - 1][i - 1] + triangle[i][i];
    }

    // 计算dp[n-1][0] 到 dp[n-1][n-1] 中的最小值
    let minTotal = dp[n - 1][0];
    for (let i = 1; i < n; ++i) {
        minTotal = Math.min(minTotal, dp[n - 1][i]);
    }
    return minTotal;
};
```

### 方法二：动态规划（自底向上做状态转移，避开了边界条件）

```javascript
 * @return {number}

  }

  // 自底而上遍历
  for (let i = n - 1; i >= 0; i--) {
    // 同层
    for (let j = 0; j < triangle[i].length; j++) {
       // base case 最底层
      if (i == n - 1) {
        dp[i][j] = triangle[i][j];
      } else {
       // 状态转移方程，上一层由它下面一层计算出
        dp[i][j] = Math.min(dp[i + 1][j], dp[i + 1][j + 1]) + triangle[i][j];
      }
    }
  }
  return dp[0][0];
};
```
